
## Install
```
pip install -r requirements.txt
```

## Usage
```
python curecon.py                              # today, all subdomains
python curecon.py --target example.com         # today, target only
python curecon.py --date 2024-03-15            # specific date, all
python curecon.py --date 2024-03-15 --target example.com
```
